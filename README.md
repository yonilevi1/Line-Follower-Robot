# Line-Follower-Robot
ESP32-based 4WD Line Follower Robot using PID control, CNY70 sensors, and L293D drivers.
# 🚗 4WD Line Follower Robot (רכב עוקב קו 4X4)

## 📌 אודות הפרויקט
פרויקט זה מציג פיתוח של רובוט אוטונומי בעל הנעה ארבע-גלגלית (4WD), המבוסס על בקר **ESP32**. הרובוט עוקב אחרי נתיב באמצעות אלגוריתם **PID** ומערך חיישנים, תוך שימוש בכוח משופר ויציבות הודות לארבעת המנועים.

## 🛠️ רכיבים וחומרה (Hardware)
הרובוט נבנה בתצורה חזקה הכוללת:

* **בקר ראשי:** ESP32 DevKit
* **חיישנים:** 3 חיישני CNY70 (מסודרים במערך לזיהוי מיקום הקו: שמאל, מרכז, ימין)
* **הנעה (Actuators):** 4 מנועי DC Gear "צהובים" (TT Motors) לתנועה בכל 4 הגלגלים.
* **דרייברים (Motor Drivers):** 2 יחידות של L293D (לשליטה נפרדת/משולבת בצד ימין ובצד שמאל ולחלוקת עומס הזרם).
* **ניהול מתח:** מייצב מתח L7805CV (להזנת הבקר והחיישנים במתח 5V יציב).

## ⚙️ לוגיקת בקרה (Control Logic)
המערכת פועלת במעגל סגור:
1.  **קלט:** שלושת חיישני ה-CNY70 מספקים קריאה דיגיטלית/אנלוגית לגבי מיקום הקו השחור.
2.  **עיבוד PID:** הבקר מחשב את השגיאה (Error) לפי שקלול החיישנים:
    * קו במרכז = שגיאה 0.
    * קו בשמאל/ימין = שגיאה חיובית/שלילית.
3.  **פלט:** תיקון מהירות המנועים. מכיוון שיש 4 מנועים, הצד השמאלי (קדמי+אחורי) והצד הימני (קדמי+אחורי) מסונכרנים כדי לפנות לכיוון הרצוי (Skid Steer).

## 🔌 סכמה חשמלית (Wiring Overview)
* **צד ימין:** 2 מנועים מחוברים לדרייבר הראשון.
* **צד שמאל:** 2 מנועים מחוברים לדרייבר השני / (או חלוקה אחרת לפי התכנון שלכם).
* **חיישנים:** מחוברים לפינים התומכים ב-ADC (ממיר אנלוגי לדיגיטלי) ב-ESP32 לקריאה מדויקת.

## 💻 התקנה והרצה
1. הורידו את הקוד.
2. ודאו התקנת מנהלי התקן ל-ESP32 ב-Arduino IDE.
3. כיילו את ערכי ה-PID בקוד (Kp, Ki, Kd) בהתאם למשטח ולמהירות המנועים.
